• Every screen: sticky white top nav (role-aware pills) → gradient header band → content cards overlapping the band → dark footer. • Nav is the shared Nav.dc.html component (props: role, active, userName). Footer is Footer.dc.html. • Spacing scale 4/8/12/16/20/24/32. Cards: white, 1px #E6E2D8, 16–18px radius; overlapping cards get the deep 0 8px 24px shadow. • Decorative language: soft translucent circles on gradient/dark bands, emoji-in-chip icons, category gradient panels — no stock photography required (image slots accept real photos). • Dates DD/MM/YYYY, 24-hour times, Singapore English throughout. • Implementation: Bootstrap 5 theme overrides (CSS variables for colours/radius) + EJS partials for navbar, header band, cards and badges.

Layout & responsive notes

• Public pages use a white top navigation bar; signed-in screens use a 248px charcoal sidebar with the user card pinned to the bottom. • On mobile, the sidebar collapses to a bottom tab bar and the top nav to a hamburger sheet — see the mobile mockups. • Spacing scale: 4 / 8 / 12 / 16 / 20 / 24 / 32. Cards: white, 1px #E6E2D8 border, 14px radius, subtle shadow. • Dates DD/MM/YYYY, 24-hour times, Singapore English throughout. • Implementation: works as Bootstrap 5 theme overrides (custom CSS variables for colours + radius) with EJS partials for navbar, sidebar, cards and badges.
